## :toolbox: Getting Started

### :bangbang: Prerequisites

- Node, npm
- StepZen
```bash
npm install -g stepzen
```


### :gear: Installation

Installation
```bash
npm install
```
Start the StepZen Server
```bash
cd api
```

```bash
stepzen start
```
Start the expo app
```bash
npm start
```
